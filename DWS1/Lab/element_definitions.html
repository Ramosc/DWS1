Definitions

Headings
Heading are displayed as < h1 >, < h2 >, < h3 >, < h4 >, < h5 >. The headings show the hierarchy of the web page.
Paragraph
Displayed < p >. This displays the text on the HTML page.
Bold
Displayed < b >. This converts word into bold styling.
Italic
Displayed < i >. This converts word into italic styling.
Superscript
Displayed < sup >. This contains element that should be superscript. Example: 102
Subscript
Displayed < sub >. This contains element that should be subscript. Example: CO2
Line Breaks
Displayed < br / >. This causes the line to break and create a new line just as the enter key would.
Horizontal Rule
Displayed < hr / >. This separates themes on the browser with a horizontal line
Strong
Displayed < strong >. This shows that this the element has a strong importance
Emphasis
Displayed < em >. This shows that this element should be emphasized and had meaning.
Blockquote
Displayed < blockquote >. Is used for long quotes.
Quote
Displayed < q >. This is use to add quotation to a sentence.
Abbreviation & Acronym
Displayed < abbr title = "" > and < acronym title = "" >. This creates the abbriviation or acronym but displays the meaning when hovered over the element.
Citation
Displayed < cite >. Creates a citation to reference outside material.
Definitions
Displayed < dfn >. Used when defining a new term for the first time.
